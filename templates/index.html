About This Tool

Our advanced comparison tool analyzes two CSV test result files and generates comprehensive reports on:

  • Tests that passed/failed in each file with smart comparison
  • Tests with "minor" bug notes highlighted
  • Detailed differences between the two result sets
  • Searchable and filterable data tables for easy navigation

Required CSV Format

Your CSV files should include these columns:
